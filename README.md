1️⃣ 프로젝트 제목
# Study GPT
초심자를 위한 학습 시작 AI Agent


2️⃣ 프로젝트 소개

## 📌 프로젝트 소개

Study GPT는 학습을 처음 시작하는 초심자를 위해  
학습 주제를 입력하면 기초 커리큘럼을 자동 생성하고,  
이해도 기반 학습과 반복 학습을 제공하는 AI Agent입니다.

기존 학습 서비스와 달리  
"어디서부터 시작해야 할지 모르는 문제"를 해결하는 데 초점을 맞췄습니다.


3️⃣ 핵심 기능
## 🚀 주요 기능

- 학습 주제 입력 및 카테고리 자동 분류
- 기초 단계 중심 커리큘럼 생성
- 학습 진행도 관리
- 퀴즈 기반 이해도 평가
- 일정 점수 미달 시 다음 단계 진행 제한
- 복습 시스템 (틀린 문제 재출제)
- 학습 요약 기능 (노트 형태)
- 사이드 질문 채팅 기능


4️⃣ 프로젝트 특징 (차별화 포인트 ⭐)

## 💡 프로젝트 특징

- 초심자를 위한 학습 시작 지원 시스템
- 기초 단계 집중 설계 (입문 ~ 기초)
- 학습 진행도 및 상태 관리
- 점수 기반 학습 통과 시스템
- 반복 학습 구조 (복습 시스템)
- 확장 가능한 구조 (RAG 기반 자료 학습 가능)
