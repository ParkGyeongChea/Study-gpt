1. LangChain
(AI 기능들을 연결하고 관리하기 쉽게 만드는 도구)

지금 현재 구조

현재는 if "다음"  같은 조건문 흐름 많음.

LangChain 들어가면

예:

memory 연결
tool 연결
RAG 연결
검색 연결

이런 것들을 체계적으로 관리 가능해짐.

쉽게 말하면 현재 기능 하나하나 직접 연결중.

LangChain 이후 AI 기능들을 조립해서 사용하는 느낌이 됨.

왜 필요한가?

프로젝트 규모 커질수록 기능 관리 난이도 폭증 하기 때문.

LangChain은 그걸 정리해주는 역할.


2. LangGraph

(AI가 상황을 보고 스스로 흐름을 판단하게 만드는 시스템)

현재는 정해진 흐름 느낌

예:

다음
다시
퀴즈

전부 if문 기반. 

LangGraph 이후 AI가 현재 상태(state) 를 보고 스스로 판단 가능.

예:

사용자 상태 분석 후:

설명 → 퀴즈 → 복습 → 다음 step 자동 판단.

가능한 기능
학습 흐름 유지
multi-step reasoning
학습 memory
사용자 수준 적응
반복 학습 자동 판단

쉽게 말하면

현재는 "버튼 누르면 정해진 반응"
LangGraph 이후
"AI가 현재 상황을 보고 행동 결정" 가능.
